# Hangman game
## Project idea
  -  This project is built in Java using the concept of OOP.
  - Inspired by the MVC design pattern, the game was separated into different classes:
    - Model i.e GameLogic
    - View i.e InputDisplay, OutputDisplay
    - Controller i.e Mediator
